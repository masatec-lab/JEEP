# Jeepping Travel Адыгея

Сайт компании по джиппинг-турам в Адыгее.

## Стек

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS 4

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)
